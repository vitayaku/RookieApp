
# Rookie App

Данный проект предназначен для проведения стажировки системных аналитиков в компании ITQ. Проект представляет из себя REST контроллер со Swagger и подключенной БД SQLite.

Для работы с проектом необходимо установить следующее ПО:

1. Visual Studio
2. SQLite Studio
3. Postman
4. Soap UI
5. Kafka
